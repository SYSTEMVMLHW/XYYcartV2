# XYYcart

本项目为一个前端产品选购的纯HTML页面，仅适用于魔方财务财务系统使用

## 使用方法
1. 修改对应文件内的配置信息
- 1.[logo地址](index.html)
- 2.[api地址](script.js)
- 3.[返回主页按钮](index.html)

2.魔方分组配置
- 1.大分组：分组名,介绍
- 2.小分组：国家缩写^分组名,介绍  或者  emoji编码|分组名,介绍
